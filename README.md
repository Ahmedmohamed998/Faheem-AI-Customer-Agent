# Faheem | AI Customer Agent 🤖

> Intelligent, multi-channel AI conversational agent for customer support, intent recognition, and automated sales lead qualification — built for [Faheem](https://faheem.app/).

## Overview

An always-on AI customer success pipeline that seamlessly integrates with WhatsApp, Instagram, and Facebook Messenger. It understands customer intent in real-time, processes daily inquiries using an advanced Retrieval-Augmented Generation (RAG) system, effortlessly recommends products, and automatically hands over complex conversations to human agents via an interactive operations dashboard.

---

## Features

- **Conversational Sales & Support** — Engages customers natively in Arabic and English. Governed by a strict engineered persona ("Faheem"), the AI never hallucinates prices or policies. It draws answers solely from injected organizational context.
- **Dynamic Knowledge Ingestion (RAG)** — A specialized Python ingestion pipeline processes PDFs, Word documents, manual entries, and structured product catalogs. It chunks and embeds text directly into a high-performance PostgreSQL [pgvector] database.
- **Smart Human Escalation** — When an inquiry surpasses the AI's contextual boundaries, it smoothly triggers an escalation protocol ("you will be redirect to human"), maintaining customer satisfaction and immediately notifying live agents.
- **Multi-Channel Asynchronous Gateway** — A unified Node.js webhook consumption engine handling thousands of concurrent interactions from Meta’s core platforms, backed by a persistent PostgreSQL session layer.
- **Live Operations Dashboard** — Employs robust WebSockets (`Socket.io`) enabling administrators and business owners to monitor conversations as they happen, customize AI responses, and manage human-handovers dynamically.

## Core AI & Data Pipelines

### 1. High-Precision RAG Engine
The decoupled Python FastAPI layer implements a continuous knowledge-ingestion pipeline. Organizations can effortlessly upload product catalogs or policy PDFs. The engine runs these through specialized text chunking utilities, vectorizes them, and persists them via SQLAlchemy using pure SQL text queries containing optimized cosine distance lookups (`<=>`) directly against PostgreSQL vector indexes. 

### 2. Defensive Persona Prompting
To mitigate enterprise AI risks (e.g., unauthorized commitments or hallucinated guarantees), the model operates under severe guardrail instructions. Running on Azure's `gpt-5-mini`, the AI is explicitly forced to output plain text (stripping markdown styling), severely capping output lengths, and is aggressively prevented from assuming facts outside the vector retrieval window. Unanswerable queries auto-detect and activate programmatic safety overrides.

### 3. Robust Multi-Channel Consumption
The Node.js Express core is engineered to concurrently process massive webhook streams generated by Meta’s ecosystems. It dictates all conversational state management, offloading intense text-embedding and language tasks to the Python worker, while leveraging Socket.io to push sub-second readouts and critical escalation warnings directly to live administration dashboards.
